Floor Plan Polygon Drawing Tool

Click on the grid to place points and create a polygon. Click near the first point to close the shape.

After committing a polygon, you'll be prompted to add a label. You can then start drawing another polygon.

You can upload a floor plan image as a background to trace over.

To edit a polygon, click on it to select it. Red handles will appear at each vertex. Drag these handles to reshape the polygon. Click anywhere outside the polygon to commit your changes.

When a polygon is selected, you can click the "Delete Polygon" button to remove it.

Use the Define Scale button to set the real-world scale of your floor plan. Draw a horizontal or vertical line and specify its length in feet.

Use the Add Furniture button to add furniture or other objects to your floor plan. You'll need to define the scale first, then specify the width and height in feet.

Use the Rotate Furniture button to rotate the currently selected object in 15-degree increments.

Use the Delete Furniture button to remove the currently selected object from the floor plan.

Use the Export Floor Grid button to save your work as a JSON file, and Import Floor Grid to load a previously saved floor plan.

Use the Show Dimensions toggle to show or hide dimensions on furniture and objects.

Use the Show Room Labels toggle to show or hide the labels on rooms and areas.
